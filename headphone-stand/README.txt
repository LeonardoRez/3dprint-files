# Summary

Simple headphone stand based on this design: 
http://www.omegaheadphonestand.com/wood-headphone-stand.html

Assembly:
You will most probably need to sand a bit the thin parts of the legs for them to fit. The gap is really tight but it will hold firmly once put together, no glue needed.

# Print Settings

Printer: Rebel 2
Rafts: Doesn't Matter
Supports: No
Resolution: 0.3
Infill: 10%

Notes: 
Material PLA
I think that infill may be set even lower, but haven't tried it yet.
It eats a lot of filament.
